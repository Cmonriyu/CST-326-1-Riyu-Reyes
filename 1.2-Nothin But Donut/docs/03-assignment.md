# Project 1 - Pong Pt. 2

Recreate the classic Pong game as specified and submit a link to your GitHub repository.

**Points:** 30

**Due:** [See Canvas]

---

## Rubric

### Working GitHub Link (−1.5 pts if missing)
If the repo link works and is accessible, you're all good.

### Active GitHub Repo (−1.5 pts if empty)
If your full project is uploaded (can be cloned, built, and run as shown in the video), you're fine.

### Requirements Completed

#### Sound Design (10 Points)
Implement sound effects when the ball collides with paddles:
- 7 pts: One sound effect for all paddle hits
- +3 pts: Sound effects change based on ball speed or paddle hit location

#### Power Ups (10 Points)
Add GameObjects to the board that alter game behavior (e.g., change ball speed, paddle size, ball direction):
- 5 pts: Implement 1 modifier
- 10 pts: Implement 2 or more modifiers

#### Scoring Display (7 Points)
Show the current score with text on a Unity Canvas:
- 5 pts: Show the incrementing score
- +2 pts: Add effects to text based on score (e.g., color changes)

#### In-Class Project (3 Points)
Complete the "Nothin But Donut" project as shown in class.

## Bonus Points (Up to 5 points)

Make Pong more "Juicy":
- +2 pts: Camera shake on collision (use Perlin noise)
- +1 pt: Impact effects when ball collides
- +1 pt: AI control for one paddle
- +1 pt: Googly-eyes that always look toward the ball
- +1 pt: Change background after each score (black → grid → texture pattern → etc)
- +1 pt: Time slow down or speed up based on gameplay
- +1 pt: Custom shader effects
- +1 pt: High score displayed and remembered between sessions
- +1 pt: Your own creative juice
